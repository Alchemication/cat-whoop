# Cat Whoop!!!

Simple Node, AngularJS and Sockets Multi-Player Game

### Demo
Demo can be played at: [demo link](https://cat-whoop.herokuapp.com/).

### Installation
  - install Node
  - clone this repo or download it
  - run `npm install`
  - run `npm start`
  - open http://localhost:3001 in the browser
  - have fun!!!
